Space Race:
This game is a simple galaga clone I helped work on for a college project.
The objective is to survive the enemies as long as possible and build up your score by killing them.
I was responsible for 
  - enemy movement (enemies moving on/off screen, diving to attack player, looping in the middle of screen, moving to position in lanes)
  - wave logic (the number of enemies that appear in a wave randomly 1-5, locking lanes when occupied so new waves don't try to move to an already occupied lane)
  - stage logic (number of enemies to spawn into a stage, modifiers that increase enemy amount/speed with every stage, pauses between stages)
